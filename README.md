# Kanban Task Board

A simple Kanban-style task management web app built using HTML, CSS, and JavaScript.

## Features

- Create tasks with title and description
- Drag and drop tasks between columns
- Three columns:
  - Todo
  - In Progress
  - Done
- Delete tasks
- Task counter for each column
- Data stored in browser using localStorage
- Tasks remain saved after page refresh
- Modal popup for adding new tasks

## Technologies Used

- HTML
- CSS
- JavaScript (DOM Manipulation)
- LocalStorage API
- Drag and Drop API

## How it works

1. User adds a task using modal form
2. Task is created dynamically using JavaScript
3. Tasks can be dragged between columns
4. Task data is saved in localStorage
5. When page reloads, tasks are restored automatically
